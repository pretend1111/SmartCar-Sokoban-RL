# Phase 1

请在 `maps/phase1/` 目录下生成 1000 张**保证可解**的推箱子地图文件（phase1_0001.txt 到 phase1_1000.txt）。

## 关键要求：必须验证可解性

生成地图后必须用项目中的 `AutoPlayer` 求解器验证每张地图确实可以通关。不可解的地图必须丢弃重新生成。

## 实现方式

请编写一个 Python 脚本，流程如下：
1. 调用 `rl/map_generator.py` 中的 `generate_map()` 函数生成地图
2. 用 `engine.py` 的 `GameEngine` 加载这张地图
3. 用 `solver/auto_player.py` 的 `AutoPlayer` 求解
4. 如果 `state.won == True` → 保存为 .txt
5. 如果未通关 → 丢弃，换一个种子重新生成
6. 重复直到凑够 1000 张可解地图

## 本阶段参数

- 箱子数：**1**，目的地数：**1**
- 炸弹数：**0**
- 内墙：**无**（`wall_density=0.0`）

## 地图格式

- 纯文本 `.txt`，12 行 × 16 字符，UTF-8，LF 换行
- 字符集：`#`=墙壁 `-`=空地 `$`=箱子 `.`=目的地 `*`=炸弹

## 脚本参考骨架

```python
import os, sys, random
ROOT = os.path.dirname(os.path.abspath(__file__))
sys.path.insert(0, ROOT)

from rl.map_generator import generate_map
from config import GameConfig
from engine import GameEngine
from solver.auto_player import AutoPlayer

cfg = GameConfig()
cfg.render_mode = "simple"
cfg.control_mode = "discrete"

out_dir = os.path.join(ROOT, 'maps', 'phase1')
os.makedirs(out_dir, exist_ok=True)

saved = 0
seed = 10001
while saved < 1000:
    map_str = generate_map(n_boxes=1, n_bombs=0, wall_density=0.0, seed=seed)
    seed += 1
    if map_str is None:
        continue
    # 写入临时文件
    tmp = os.path.join(out_dir, '_tmp.txt')
    with open(tmp, 'w', encoding='utf-8') as f:
        f.write(map_str)
    # 验证可解性
    engine = GameEngine(cfg, ROOT)
    random.seed(42)
    engine.reset(os.path.relpath(tmp, ROOT).replace('\\\\', '/'))
    player = AutoPlayer(engine)
    actions = player.solve()
    if actions:
        random.seed(42)
        engine.reset(os.path.relpath(tmp, ROOT).replace('\\\\', '/'))
        for a in actions:
            engine.discrete_step(a)
        if engine.get_state().won:
            saved += 1
            dest = os.path.join(out_dir, f'phase1_{saved:04d}.txt')
            os.rename(tmp, dest)
            if saved % 100 == 0:
                print(f'已生成 {saved}/1000')
            continue
    os.remove(tmp)
print(f'完成! 共生成 {saved} 张可解地图')
```

请直接在项目根目录运行此脚本。先清空 `maps/phase1/` 中已有的旧文件再生成。
